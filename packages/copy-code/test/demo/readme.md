## 使用插件

### 安装

```bash
npm i -D @mr-hope/vuepress-plugin-copy-code
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugin: [
    "@mr-hope/copy-code",
    {
      // 你的配置选项
    },
  ],
};
```
